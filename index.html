The Power Of Brackets Beautification!!!!!!

Extension

You can find this extension in 'Extension Manager' in brackets, which is located at the right hand side of the screen.You have to type 'Beautify'in the search box and install it.

Uses

In Brackets, the beautify extension is useful in ... beautificating the code and making easy to read and understand. You could write the entire code in 1 line, click beautify and have it space everything out!
